# Portfolio - Davi Oliveira

Portfolio profissional desenvolvido com Django, apresentando projetos, competencias, experiencias e formacao de um Desenvolvedor Backend Python.

Acesse: [davioliveira.tech](https://davioliveira.tech)

---

## Finalidade

Construir um portfolio pessoal completo e funcional, desde o desenvolvimento ate o deploy em producao, aplicando boas praticas de engenharia de software em todas as etapas do projeto.

---

## Tecnologias Utilizadas

- **Backend:** Python, Django
- **Frontend:** HTML5, CSS3, JavaScript
- **Banco de Dados:** PostgreSQL
- **Servidor:** Gunicorn, Nginx
- **Infraestrutura:** VPS Hostinger (Ubuntu 24.04), SSL com Let's Encrypt
- **Qualidade:** Pytest, Black, isort, Pre-commit
- **Versionamento:** Git, GitHub

---

## Conceitos Aprendidos

- Arquitetura de projetos Django com separacao em apps
- Configuracao de ambientes de desenvolvimento e producao
- Formularios com validacao via ModelForm
- Envio de emails com SMTP (Gmail)
- Deploy completo em VPS com Gunicorn + Nginx + systemd
- Configuracao de SSL/TLS com Certbot
- Servicos de arquivos estaticos com WhiteNoise
- Testes automatizados com Pytest e cobertura de 100%
- Pre-commit hooks para padronizacao de codigo
